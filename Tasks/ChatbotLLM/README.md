# ChatbotLLM — Local Oriental Restaurant Chatbot

A console-based chatbot powered by a local GGUF language model via **llama-cpp-python**.  
The bot acts as a virtual assistant for a restaurant.

---

## Prerequisites

- **Python 3.10+**
- **llama-cpp-python** and **PyYAML** (listed in `requirements.txt`)
- A **GGUF** language model file (e.g., from [Hugging Face](https://huggingface.co/))

---

## Setup

1. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Place your GGUF model** in the `models/` directory;

---

## Running

```bash
python main.py
```
